Glossary

eLearning jargon, decoded

32 terms every instructional designer should know.

ABCEFGHIJKLMNOPQRSTUVWXZ

A

ADDIE
The five-phase instructional design model: Analyse, Design, Develop, Implement, Evaluate. Best for well-scoped projects.
Andragogy
The art and science of adult learning, popularised by Malcolm Knowles. Adults bring experience, want relevance, and learn best when self-directed.
Authoring Tool
Software used to create eLearning content — Articulate Storyline, Rise, Captivate, Lectora, DominKnow ONE, etc.

B

Bloom's Taxonomy
A hierarchical model of cognitive skills: Remember → Understand → Apply → Analyse → Evaluate → Create.
Branching Scenario
An interactive activity where learner choices lead to different consequences and learning paths.

C

Captivate
Adobe's eLearning authoring tool known for software simulations and responsive output.
Cognitive Load
The mental effort required to learn. Sweller's theory: intrinsic + extraneous + germane load.
Constructivism
The theory that learners construct knowledge by linking new info to existing schemas.

E

eLearning
Any learning delivered electronically — web, mobile, VR, or LMS-based courses.

F

Flesch Reading Ease
A 0–100 readability score. Higher is easier. Aim for 60–80 for workplace eLearning.
Formative Assessment
Low-stakes checks-for-understanding embedded during learning, not at the end.

G

Gamification
Applying game mechanics (points, levels, badges) to non-game contexts. Different from a full game.

H

Heutagogy
Self-determined learning — the learner chooses the topic, pace, and assessment.

I

Instructional Design
The systematic practice of designing learning experiences. Combines learning science, UX, and project management.

J

Just-in-Time Learning
Performance support delivered at the moment of need — a 30-second video on how to reset a router.

K

Kirkpatrick Model
Four levels of training evaluation: Reaction → Learning → Behavior → Results.

L

Learning Management System (LMS)
Software that hosts, delivers, tracks, and reports on training — HealthStream, Moodle, Docebo, etc.

M

Microlearning
Short, focused, single-objective learning bursts — typically 3–7 minutes.
Mayer's Principles
Twelve research-backed multimedia principles. Includes contiguity, coherence, signalling, segmentation.

N

Needs Analysis
Diagnosing the performance gap before designing training. Often skipped — and badly.

O

Objectives
Specific, measurable statements of what learners will be able to do. Use action verbs, not "understand".

P

Pedagogy
Originally child-teaching, now sometimes used for any instructional method.

Q

Quiz
A short assessment, typically formative. Can also be summative when graded.

R

Rise 360
Articulate's web-based responsive authoring tool. Block-based and mobile-first.

S

SCORM
Sharable Content Object Reference Model — the most common eLearning standard for LMS interoperability.
Storyboard
A document describing every screen of a course before it's built — text, visuals, narration, interactions.

T

Taxonomy
A classification system. In learning: Bloom's, SOLO, Marzano, etc.

U

Universal Design for Learning (UDL)
A framework that designs for the widest range of learners from the start — not retrofitting accessibility.

V

Vyond
A cloud-based animation tool useful for character-driven explainer videos.

W

WCAG
Web Content Accessibility Guidelines — currently 2.2. AA is the working standard for most enterprise eLearning.

X

xAPI
Experience API — the successor to SCORM. Captures any learning experience as a JSON statement.

Z

Z-Pattern
A visual layout pattern that mirrors how Western readers scan a page in a Z shape.
