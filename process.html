My Process

From messy brief to measurable learning

A simple, repeatable, 5-stage process I've refined across 80+ projects.

01
🔍
Week 1

Discover

I start by listening — to stakeholders, SMEs, and the people who'll actually take the course.

  • ✓ Kick-off & goals workshop
  • ✓ Audience & persona profile
  • ✓ Content audit
  • ✓ Success-metrics defined
  • ✓ Risk & dependency register
02
✏️
Week 2–3

Design

Before a single Storyline slide is built, we agree on the story.

  • ✓ Course outline & learning objectives
  • ✓ Detailed storyboard
  • ✓ Visual & motion concept
  • ✓ Interactive prototype (1 module)
  • ✓ Sign-off gate
03
⚙️
Week 3–6

Develop

Where the magic — and the JavaScript — happens.

  • ✓ Authoring in Storyline / Rise / Captivate
  • ✓ Custom JS interactions
  • ✓ Voice-over recording
  • ✓ WCAG 2.1 AA pass
  • ✓ Alpha & beta review cycles
04
🚀
Week 7

Deliver

Ship it. Track it. Validate it.

  • ✓ SCORM 1.2 / 2004 / xAPI packaging
  • ✓ LMS deployment & smoke test
  • ✓ Learner pilot (10–20 people)
  • ✓ Launch comms support
  • ✓ Post-launch QA
05
📈
Ongoing

Measure & Iterate

Done is the start, not the finish.

  • ✓ xAPI dashboards live in SharePoint
  • ✓ Quarterly learner feedback loop
  • ✓ Content refresh cadence
  • ✓ Performance metrics review
  • ✓ Roadmap for v2
