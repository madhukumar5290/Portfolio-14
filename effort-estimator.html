How to use this number

These estimates are based on Chapman Alliance and ATD benchmarks for finished-minute-to-development-hour ratios, weighted by complexity and interactivity.

Use them for: sanity-checking SME estimates, proposal pricing, sprint planning.

Don't use them for: the only number in your project plan. Add 15% buffer minimum for review cycles, voice-over, and translation.
