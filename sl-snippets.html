Storyline JS Snippets

8 snippets worth bookmarking

Drop-in JavaScript for Articulate Storyline 360 — copy, paste, ship.

Snippet · JavaScript

Get a Storyline variable

Pull a Storyline variable into JS for any custom logic.

var player = GetPlayer();
var name = player.GetVar('UserName');
console.log('Hi ' + name);
Snippet · JavaScript

Set a Storyline variable

Push a value from JS back into a Storyline variable.

var player = GetPlayer();
player.SetVar('UserScore', 92);
Snippet · JavaScript

Random number

Generate a random integer between min and max.

function rnd(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min;
}
var player = GetPlayer();
player.SetVar('Dice', rnd(1, 6));
Snippet · JavaScript

Trigger a slide jump

Send the learner to a specific slide from JS.

// Set a Storyline trigger to jump when this var changes
var player = GetPlayer();
player.SetVar('GoToSlide', 'Scene1.Slide5');
Snippet · JavaScript

Open a PDF in new tab

Open a resource without leaving the course.

window.open('https://example.com/policy.pdf', '_blank', 'noopener');
Snippet · JavaScript

RTL flip on language switch

Toggle right-to-left layout for Arabic/Hebrew learners.

var lang = GetPlayer().GetVar('Language');
var dir = (lang === 'ar' || lang === 'he') ? 'rtl' : 'ltr';
document.documentElement.setAttribute('dir', dir);
Snippet · JavaScript

Save progress to localStorage

Persist learner state across sessions without an LMS.

localStorage.setItem('mk_progress', JSON.stringify({
  slide: 4,
  score: 78,
  ts: Date.now()
}));
Snippet · JavaScript

Send xAPI statement

Fire a custom xAPI statement to your LRS.

var stmt = {
  actor: { mbox: 'mailto:learner@example.com' },
  verb:  { id: 'http://adlnet.gov/expapi/verbs/passed' },
  object:{ id: 'https://lms.example.com/courses/fire-safety' }
};
fetch('https://lrs.example.com/statements', {
  method: 'POST',
  headers: { 'Content-Type':'application/json' },
  body: JSON.stringify(stmt)
});
